• HTML is the standard markup language for creating Web pages.
    HTML describes the structure of a Web page.
    HTML consists of a series of elements.
    HTML elements tell the browser how to display the content.

  • CSS is a tool that helps developers.
    CSS makes "the colors" of the Web page.
    Any element in the HTML file can be designed differently.
    It is easy to use and learn.

  • JavaScript is a programming language that makes the HTML file functional.
    JavaScript is a complex language yet used everywhere.
    It can be used for Frontend or Backend development.
    JavaScript tells the elements like buttons etc. what to do.

  • React is a commonly used framework for development.
    React is mainly used in Frontend development.
    It is also described as a library used for developing user intefraces.

  • Bootstrap is a free,open sourse front-end development framework.
    It is used for developing websites and web apps.
    Bootstrap provides a collection of syntax for template design.
